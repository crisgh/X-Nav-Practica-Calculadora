Calculadora

Calculadora donde podemos realizar operaciones aritmeticas basicas ( / * - + ) con los numeros (del 0 al 9), además podemos hayar la raiz del número con "Sqrt" y el valor inverso con "1/X", dando a “=” para obtener el resultado,tenemos “C” para borrar, "Del" para eliminar el último número que hayamos introducido.
